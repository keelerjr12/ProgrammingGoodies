# ProgrammingGoodies

## Adding existing project to Github
```git init```

```git add .``` or ```git add -A```

```git commit -m "Initial Commit"```

```git push origin master```

## Track specific file
```git add [file]```

## Untrack specific file
``` git rm -r --cached [file]```

## Untrack all files
``` git rm -r --cached *```
